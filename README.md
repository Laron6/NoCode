# Приложение для создания опросов и голосования  
с сохранением в localStorage

Автор: Bogdan Matrosov  
Год: 2026

## Запуск
- Дважды кликнуть `index.html`  
- Тесты: `node --test voting.test.js`

## Сценарии тестирования

### Тест 1. Создание опроса
**Цель**: Проверить создание опроса  
**Предпосылки**: Страница открыта  
**Шаги**:  
1. Ввести заголовок (например "Лучший фильм?")  
2. Нажать "Создать"  

**Ожидаемый результат**: Сообщение "Опрос создан. ID: 1"  
**Фактический результат**: Сообщение "Опрос создан. ID: 1" появилось, опрос отобразился в списке

### Тест 2. Добавление пользователя
**Цель**: Регистрация пользователя  
**Шаги**:  
1. Ввести ID (user1) и имя (Богдан)  
2. Нажать "Добавить"  

**Ожидаемый результат**: Сообщение "Пользователь добавлен"  
**Фактический результат**: Сообщение "Пользователь добавлен" появилось, пользователь отображается в списке

### Тест 3. Голосование
**Цель**: Успешное голосование и запрет повторного  
**Предпосылки**: Опрос и пользователь созданы  
**Шаги**:  
1. Ввести ID опроса, вариант (0), свой ID  
2. Нажать "Голосовать"  
3. Повторить шаг 1  

**Ожидаемый результат**: Первый — успех, второй — ошибка "уже голосовал"  
**Фактический результат**: Первый голос принят (сообщение "Голос учтён"), при повторной попытке появилось сообщение "Вы уже голосовали в этом опросе"

### Тест 4. Просмотр результатов
**Цель**: Корректные результаты после голосов  
**Шаги**:  
1. Ввести ID опроса  
2. Нажать "Результаты"  

**Ожидаемый результат**: Отображается заголовок, голоса по вариантам, общее количество  
**Фактический результат**: Отображается корректный заголовок опроса, список вариантов с количеством голосов и общее число проголосовавших

### Тест 5. Удаление опроса
**Цель**: Удаление и очистка голосов  
**Шаги**:  
1. Ввести ID опроса  
2. Нажать "Удалить"  

**Ожидаемый результат**: Сообщение "Опрос удалён", результаты недоступны  
**Фактический результат**: Появилось сообщение "Опрос удалён", опрос исчез из списка, при попытке посмотреть результаты появляется сообщение об ошибке / опрос не найден

### Тест 6. Сохранение после перезагрузки
**Цель**: Данные не теряются  
**Шаги**:  
1. Создать опрос и проголосовать  
2. Перезагрузить страницу  
3. Посмотреть результаты  

**Ожидаемый результат**: Всё на месте  
**Фактический результат**: После перезагрузки все созданные опросы, пользователи и отданные голоса сохранились и корректно отображаются

## Очистка данных
Кнопка "Очистить все данные" → подтверждение → localStorage очищается.  
После очистки: список опросов пуст, пользователи удалены, все данные сброшены.

Все базовые сценарии протестированы и работают корректно.
