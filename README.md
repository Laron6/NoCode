# Library Management System

Система управления библиотекой: добавление книг и читателей, выдача/возврат книг конкретным пользователям, отслеживание кто что взял. Всё сохраняется в localStorage.

Автор: Bogdan Matrosov  
Год: 2026

## Запуск
- Открой `index.html` в браузере  
- Тесты: `node --test library.test.js`

## Сценарии тестирования

### Тест 1. Добавление книги
**Цель**: Проверить добавление новой книги  
**Шаги**:  
1. Ввести название и (опционально) автора  
2. Нажать «Добавить книгу»  
**Ожидаемый результат**: Книга появляется в списке с зелёной меткой «Доступна»  
**Фактический результат**: Книга добавлена, отображается корректно, сохраняется после перезагрузки

### Тест 2. Добавление читателя
**Цель**: Регистрация нового читателя  
**Шаги**:  
1. Ввести имя  
2. Нажать «Добавить читателя»  
**Ожидаемый результат**: Читатель появляется в списке с иконкой человека  
**Фактический результат**: Читатель добавлен, количество взятых книг = 0

### Тест 3. Выдача книги конкретному читателю
**Цель**: Проверка выдачи книги от конкретного пользователя  
**Шаги**:  
1. Добавить книгу и читателя  
2. Под книгой нажать имя нужного читателя  
**Ожидаемый результат**: Книга становится жёлтой, у читателя появляется запись, имя читателя отображается у книги  
**Фактический результат**: Всё работает: книга взята именно этим читателем, статистика обновлена

### Тест 4. Возврат книги
**Цель**: Возврат книги в библиотеку  
**Шаги**:  
1. Взять книгу  
2. Нажать «Вернуть»  
**Ожидаемый результат**: Книга снова зелёная, у читателя убирается запись  
**Фактический результат**: Книга вернулась в доступные, список читателя чист

### Тест 5. Удаление книги с учётом выдачи
**Цель**: Удаление книги, даже если она взята  
**Шаги**:  
1. Взять книгу читателем  
2. Нажать «Удалить» на книге  
**Ожидаемый результат**: Книга удаляется, у читателя очищается список взятых  
**Фактический результат**: Книга удалена, у читателя больше нет этой книги

### Тест 6. Запрет удаления читателя с книгами
**Цель**: Защита от удаления активного читателя  
**Шаги**:  
1. Выдать книгу читателю  
2. Нажать «Удалить» у читателя  
**Ожидаемый результат**: Ошибка: «Читатель имеет взятые книги»  
**Фактический результат**: Ошибка появляется, читатель не удаляется

Все сценарии протестированы вручную и автоматически — приложение полностью работоспособно.
