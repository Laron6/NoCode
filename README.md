# To-Do List

Система управления задачами: добавление, редактирование, удаление, пометка выполненных. Данные сохраняются в localStorage.

Автор: Bogdan Matrosov  
Год: 2026

## Запуск
- Дважды кликнуть `index.html`  
- Тесты: `node --test todo.test.js`

## Сценарии тестирования

### Тест 1. Добавление задачи
**Цель**: Создание новой задачи  
**Предпосылки**: Страница открыта  
**Шаги**:  
1. Ввести название и описание  
2. Нажать "Добавить"  

**Ожидаемый результат**: Задача появляется в списке, сохраняется после перезагрузки  
**Фактический результат**: Задача успешно добавилась в список, после перезагрузки страницы остаётся на месте

### Тест 2. Редактирование задачи
**Цель**: Изменение названия/описания  
**Шаги**:  
1. Нажать "Редактировать" на задаче  
2. Ввести новые данные  

**Ожидаемый результат**: Задача обновляется, изменения сохраняются  
**Фактический результат**: Название и описание успешно изменились, новые значения отображаются корректно, сохраняются после перезагрузки

### Тест 3. Пометка выполненной
**Цель**: Переключение статуса  
**Шаги**:  
1. Поставить чекбокс на задаче  
2. Снять чекбокс  

**Ожидаемый результат**: Статус меняется, строка зачёркивается  
**Фактический результат**: При установке чекбокса задача зачёркивается и становится полупрозрачной, при снятии — возвращается в обычный вид, статус корректно сохраняется

### Тест 4. Удаление задачи
**Цель**: Удаление из списка  
**Шаги**:  
1. Нажать "Удалить" на задаче  
2. Подтвердить  

**Ожидаемый результат**: Задача исчезает, данные очищаются  
**Фактический результат**: После подтверждения задача исчезла из списка, при перезагрузке страницы не появляется снова

### Тест 5. Получение всех задач
**Цель**: Отображение списка  
**Шаги**:  
1. Добавить несколько задач  
2. Перезагрузить страницу  

**Ожидаемый результат**: Все задачи на месте, статистика обновлена  
**Фактический результат**: Все добавленные задачи отображаются после перезагрузки, статистика «Всего задач» показывает правильное количество

### Тест 6. Получение выполненных задач
**Цель**: Фильтр по статусу  
**Шаги**:  
1. Добавить задачи, пометить одну выполненной  
2. Проверить статистику  

**Ожидаемый результат**: Выполненных — 1, всего — 2  
**Фактический результат**: Статистика показывает «Всего задач: 2 | Выполнено: 1», выполненная задача корректно отображается зачёркнутой

## Очистка данных
Кнопка «Очистить всё» → подтверждение → все задачи удаляются из списка и из localStorage.  
После очистки список пустой, статистика показывает 0 задач.

Все основные сценарии протестированы вручную и автоматически (unit-тесты проходят успешно). Приложение полностью работоспособно.
