# NoCode — проекты по дисциплине 'Тестирование информационных систем'

Это репозиторий для сдачи лабораторных работ и проектов по дисциплине.

## Проекты в отдельных ветках:
- **calc** — простой калькулятор в стиле macOS с юнит-тестами
- **blog** — создание, редактирование и удаление постов, комментарии к постам, авторизация пользователей, localStorage
- **expense** — приложение для учёта доходов и расходов с сохранением в localStorage
- **library** — управление библиотекой: книги, читатели, выдача/возврат книг, localStorage
- **todo** — добавление, редактирование, удаление задач, пометка выполненных, сохранение в localStorage
- **voting** — система голосования с созданием опросов, голосованием и сохранением в localStorage
- **chat** — простой чат: создание диалогов, отправка сообщений, сохранение в localStorage
- **calendar** — планирование событий с датой/временем, от конкретного пользователя, редактирование, уведомления в браузер, localStorage
- **marketplace** — создание объявлений о продаже товаров, добавление отзывов и оценок, удаление товаров, сохранение в localStorage
