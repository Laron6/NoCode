# Торговая площадка

Приложение для создания объявлений о продаже товаров, добавления отзывов и оценок, удаления товаров. Данные сохраняются в localStorage.

Автор: Bogdan Matrosov  
Год: 2026

## Запуск
- Дважды кликнуть index.html  
- Тесты: `node --test marketplace.test.js`

## Сценарии тестирования

### Тест 1. Добавление объявления
**Цель**: Создание нового товара  
**Шаги**:  
1. Ввести название, цену, описание  
2. Нажать "Опубликовать"  

**Ожидаемый результат**: Объявление появляется в списке, сохраняется после перезагрузки  
**Фактический результат**: Товар успешно добавлен, отображается с ценой и описанием

### Тест 2. Добавление отзыва и оценки
**Цель**: Оставить отзыв на товар  
**Шаги**:  
1. Добавить товар  
2. Нажать "Оставить отзыв"  
3. Ввести текст и оценку  

**Ожидаемый результат**: Отзыв появляется под товаром, рейтинг отображается звёздами  
**Фактический результат**: Отзыв добавлен, звёзды показывают оценку

### Тест 3. Удаление объявления
**Цель**: Удаление товара из списка  
**Шаги**:  
1. Добавить товар  
2. Нажать "Удалить" под объявлением  
3. Подтвердить  

**Ожидаемый результат**: Объявление исчезает из списка  
**Фактический результат**: Товар удалён, список обновлён

### Тест 4. Сохранение после перезагрузки
**Цель**: Данные не теряются  
**Шаги**:  
1. Добавить товары и отзывы  
2. Перезагрузить страницу  

**Ожидаемый результат**: Все объявления и отзывы на месте  
**Фактический результат**: Всё сохранилось, отображается корректно

### Тест 5. Краевой случай — некорректный ввод
**Цель**: Защита от пустых данных  
**Шаги**:  
1. Оставить поля пустыми и нажать "Опубликовать"  

**Ожидаемый результат**: Появляется предупреждение, товар не добавляется  
**Фактический результат**: Ошибка отображается, ничего не добавляется

### Тест 6. Добавление нескольких отзывов
**Цель**: Проверка нескольких отзывов на один товар  
**Шаги**:  
1. Добавить товар  
2. Оставить 2–3 отзыва  

**Ожидаемый результат**: Все отзывы отображаются под товаром  
**Фактический результат**: Отзывы добавлены, отображаются корректно

## Очистка данных
Очистка через `clearAll()` в коде или перезапись localStorage → все товары и отзывы удаляются.  
После очистки список пустой.

Все основные сценарии протестированы вручную и автоматически (unit-тесты проходят успешно). Приложение полностью работоспособно.
