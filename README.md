# Блог-платформа

Простая блог-платформа: создание, редактирование, удаление постов, добавление комментариев. Данные сохраняются в localStorage (имитация RESTful API).

Автор: Bogdan Matrosov  
Год: 2026

## Запуск
- Дважды кликнуть index.html  
- Тесты: `node --test blog.test.js`

## Сценарии тестирования

### Тест 1. Создание поста
**Цель**: Публикация нового поста  
**Шаги**:  
1. Ввести заголовок и текст  
2. Нажать "Опубликовать"  

**Ожидаемый результат**: Пост появляется в списке, сохраняется после перезагрузки  
**Фактический результат**: Пост успешно создан и отображается

### Тест 2. Добавление комментария
**Цель**: Оставить комментарий к посту  
**Шаги**:  
1. Создать пост  
2. Нажать "Добавить комментарий"  
3. Ввести текст  

**Ожидаемый результат**: Комментарий появляется под постом  
**Фактический результат**: Комментарий добавлен корректно

### Тест 3. Удаление поста
**Цель**: Удаление поста из блога  
**Шаги**:  
1. Создать пост  
2. Нажать "Удалить пост"  
3. Подтвердить  

**Ожидаемый результат**: Пост исчезает из списка  
**Фактический результат**: Пост удалён, список обновлён

### Тест 4. Сохранение после перезагрузки
**Цель**: Данные не теряются  
**Шаги**:  
1. Создать посты и комментарии  
2. Перезагрузить страницу  

**Ожидаемый результат**: Все посты и комментарии на месте  
**Фактический результат**: Всё сохранилось, отображается корректно

### Тест 5. Краевой случай — пустой пост
**Цель**: Защита от некорректного ввода  
**Шаги**:  
1. Оставить поля пустыми и нажать "Опубликовать"  

**Ожидаемый результат**: Появляется предупреждение, пост не добавляется  
**Фактический результат**: Ошибка отображается, ничего не добавляется

### Тест 6. Несколько комментариев к посту
**Цель**: Проверка нескольких комментариев  
**Шаги**:  
1. Создать пост  
2. Добавить 2–3 комментария  

**Ожидаемый результат**: Все комментарии отображаются под постом  
**Фактический результат**: Комментарии добавлены, отображаются корректно

## Очистка данных
Очистка через `clearAll()` в коде или перезапись localStorage → все посты и комментарии удаляются.  
После очистки список пустой.

Все основные сценарии протестированы вручную и автоматически (unit-тесты проходят успешно). Приложение полностью работоспособно.
